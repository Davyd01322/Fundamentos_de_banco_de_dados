Quem usa o banco de dados?
  - Administrador;
  - Projetistas de BD;
  - Desenvolvedores;
  - Analistas de BD;
  - Usuários Finais;

Operações com banco de Dados:
  Definição: Tipos, estruturas e restrições dos dados;
  Construção: Criar o banco, definir estruturas de armazenamento;
  Manipulação: Envolve consultas e atualizações;

Linguages do SGBD:
  A SQL(Structured Query Language) é a linguagem padrão usada para interagir com banco de dados relacionais;

Linguagens do SGBD
  Linguagem de definição de dados (DDL - Data Definitinion Language): É como se fosse um subconjunto do SQL.
  Linguagem de Manipulação de dados (DML - Data Manipulation Language):
      ex.:
      UPDATE ALUNO SET NAME = "JOÃO COSTA" WHERE MATRICULA = 101;
      DELETE FROM ALUNO;
  Linguagem de manipulação de dados (DQL - Data Query Language):
      SELECT * FROM ALUNO;

Ambiente de um sistemea de banco de dados:
  Software SGBD: No nosso caso será o PostgreSQL que por sua vez processa e acessa os dados.
  Vamos acessar o banco de dados através de uma aplicação/consulta.

Arquitetura física centralizada

Arquitetura cliente-servidor
  - 2 camadas:
      Uma máquina que roda uma aplicação cliente que se conecta com um servidor que mantém o banco de dados.
  - 3 camadas:
      Interface web, Servidor de aplicação, servidor de BD e o próprio banco de dados.

Exemplo de uma transação feita via SQL:
  START TRANSACTION;
  UPDATE contas SET saldo = saldo - 100 WHERE id = 1;
  UPDATE contas SET saldo = saldo + 100 WHERE id = 2;
  COMMIT;
